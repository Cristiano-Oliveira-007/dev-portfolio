Aula #03 - Tipos de dados 

- typeof - Retona o tipo de valor

Ex: typeof(8) = 'number'
    typeof(8.2) = 'number'
    typeof('8') = 'string'
    typeof("8") = 'string'

let fruta = 'maçã';
typeof(fruta) = 'string'

let primeiroNumero = 5
typeof(primeiroNumero) = 'number'

let animal
typeof(animal) = 'undefined' (Isso ocorre pois o valor de 'let animal' 
não foi definido)

Valores declaredos dentro de "" ou '' são considerados string (texto)
(Em alguns casos)

Ex: let verdadeiro = true;
    typeof(verdadeiro) = 'boolean'

parseInt = parse (Converter) Int (tipo de dado)

parseInt - valor inteiro
Ex: 1

parseFloat - Valor flutuavél
Ex: 1,1

let primeiroNumero = '5'
typeof(primeiroNumero) = 'string'
typeof(parseInt(primeiroNumero)) = 'number'

parseInt(primeiroNumero) = 5
parseInt('56') = 56

parseInt('nome') = NaN (NOT A NUMBER)
